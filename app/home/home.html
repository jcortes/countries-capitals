This is a tool that allows users to search and visualize data about countries and capital cities

Browse Countries: This functionality lets the user see all of the countries about which they can view more detailed information. Each row entry has the following summary information: name, country code, capital, area in square kilometers, population and continent code

When you click on a row entry, you should be taken to the country detail view for that country.

Country Detail: This functionality shows users detailed information about a country. The following data should be displayed: country name, population of country, area, capital, population of capital, continent, timezone and number and names of neighbors.
